📦 Variables

What is a Variable?

A variable is a container used to store data. Think of it like a box where we keep values.

let student = "Ravi";

Variable Declaration

We can declare variables using: • var (old method) • let (modern, changeable) • const (fixed value)

let age = 20;

Naming Rules

✔ Can use letters, $, _ ✔ Must be meaningful ❌ Cannot start with number

studentName ✔

Scope

Scope defines where a variable can be used. • Global → anywhere • Local → inside function

function test(){ let x = 10; }

Reassignment

We can change value of let variables. Used in dynamic systems like seats.

let seats = 50; seats = 49;
🔢 Data Types

What are Data Types?

Data type defines what kind of value a variable holds. Example: number, text, true/false

Primitive Types

Basic single values: • String • Number • Boolean • Null • Undefined

let marks = 90;

Non-Primitive Types

Stores multiple values: • Array • Object

let arr = ["CSE", "ECE"];

Type Checking

We use typeof to check data type.

typeof 10 → number

Type Conversion

Converting one type to another: String → Number or Number → String

Number("20") → 20
